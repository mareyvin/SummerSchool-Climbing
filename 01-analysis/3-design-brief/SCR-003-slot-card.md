# Требования на дизайн · SCR-003 · Карточка тренировки

**ID:** SCR-003 · **Тип:** Экран · **Зона:** АЗ · **Приоритет:** Critical · **Статус:** Черновик · **Версия:** 0.1 · **Дата:** 2026-07-04

**Источники:** [Foundations](00-foundations.md) · [ФТ](../functional-requirements.md) · [Use cases](../use-cases.md)

## 1. Назначение и контекст
Полная информация об одной тренировке перед принятием решения о записи. Последний экран перед оформлением брони (UC-1).

## 2. Навигация
### Входящие
- Карточка слота на [SCR-002](SCR-002-slot-list.md).
### Исходящие
- CTA «Записаться» → [SCR-004](SCR-004-booking.md) (доступен только при наличии мест и права доступа).
- «Назад» → SCR-002.

## 3. Контент-инвентарь
| # | Элемент | Источник | Назначение |
|---|---|---|---|
| 1 | Дата/время старта, длительность | `slot.start_at`, `slot.duration` | Когда и насколько. |
| 2 | Зона/формат + бейдж «Для новичков» | `slot.zone`, `slot.is_beginner` | Что за тренировка. |
| 3 | Инструктор, средний рейтинг | `slot.instructor.*` | Кто ведёт (FR-41); «Нет оценок», если рейтинга ещё нет. |
| 4 | Свободно / всего мест | `slot.free_seats/capacity` | Доступность группы. |
| 5 | Доступность и тариф проката снаряжения | `slot.rental_available`, `slot.rental_tariff` | Для решения о своём/прокатном снаряжении. |
| 6 | Адрес скалодрома | `gym.address` | Где. |
| 7 | CTA «Записаться» (или дисейбл-состояние) | — | Переход к оформлению. |

## 5. Компоненты и поведение
- Если `free_seats = 0` — CTA заменяется на плашку «Мест нет».
- Если слот «для новичков» и у клиента нет флага — CTA заменяется на пояснение «Тренировка только для новичков» (FR-16), без грубого отказа.
- Если слот отменён скалодромом — вместо CTA показывается блок с причиной отмены (FR-29), запись недоступна.
- Рейтинг инструктора: если оценок нет — «Нет оценок» вместо «0» (FR-41).

## 7. Критерии приёмки
```gherkin
Функция: Карточка тренировки (SCR-003)

  Сценарий: AC-001 — доступна запись
    Дано free_seats > 0 и (слот не для новичков ИЛИ у клиента есть флаг «новичок»)
    Тогда кнопка «Записаться» активна и ведёт на SCR-004

  Сценарий: AC-002 — мест нет
    Дано free_seats = 0
    Тогда вместо кнопки показана плашка «Мест нет»

  Сценарий: AC-003 — новичковый слот без флага
    Дано слот is_beginner = true, у клиента флаг «новичок» не установлен
    Тогда вместо кнопки показано пояснение о том, что тренировка только для новичков

  Сценарий: AC-004 — рейтинг без оценок
    Дано у инструктора нет ни одной оценки
    Тогда вместо числового рейтинга показано «Нет оценок»
```
