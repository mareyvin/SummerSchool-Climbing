---

## 1. Назначение и контекст
Разовая оценка инструктора после посещённой тренировки: 1–5 звёзд + необязательный комментарий (UC-4).

## 2. Навигация
- Входящая: «Оценить инструктора» на [SCR-005](SCR-005-my-bookings.md) или [SCR-006](SCR-006-booking-details.md).
- Исходящая: отправка → закрытие → экран-родитель с сообщением «Спасибо за оценку».

## 3. Контент-инвентарь
| # | Элемент | Назначение |
|---|---|---|
| 1 | Заголовок «Оцените инструктора» + имя инструктора | Контекст. |
| 2 | Шкала 1–5 звёзд (тап/клик по звезде) | Обязательный элемент оценки. |
| 3 | Поле комментария (необязательно) | Текстовая обратная связь. |
| 4 | Кнопка «Отправить» (disabled до выбора звёзд) | Подтверждение. |
| 5 | Кнопка/ссылка «Не сейчас» | Закрыть без отправки, не блокирует, можно оценить позже. |

## 4. Структура
```
┌─────────────────────────────┐
│  Оцените инструктора Анну     │
│      ☆ ☆ ☆ ☆ ☆                │  ← тап по звезде выставляет оценку
│  ┌───────────────────────┐   │
│  │ Комментарий (необяз.)  │   │
│  └───────────────────────┘   │
│  ┌───────────────────────┐   │
│  │       Отправить        │   │  ← disabled, пока не выбрана хотя бы 1 звезда
│  └───────────────────────┘   │
│           Не сейчас           │
└─────────────────────────────┘
```

## 5. Компоненты и поведение
- Звёзды — крупные тач-зоны (≥44px на мобильном, §3.1 foundations).
- После отправки оценку **нельзя изменить** (FR-40) — при повторном открытии для этой брони показывается только просмотр (см. SCR-006 §4b).
- «Не сейчас» не блокирует и не считается отказом навсегда — CTA «Оценить» остаётся на карточке брони, пока оценка не отправлена (P7, ненавязчиво).

## 6. Валидации
| Правило | Реакция |
|---|---|
| Не выбрана ни одна звезда | Кнопка «Отправить» неактивна |
| Уже оценено (`already_rated`) | Модалка не открывается повторно — на карточке сразу показан просмотр оценки |

## 7. Критерии приёмки
```gherkin
Функция: Оценка инструктора (BS-004)

  Сценарий: AC-001 — успешная отправка оценки
    Дано клиент выбрал 4 звезды и написал комментарий
    Когда нажимает «Отправить»
    Тогда оценка сохраняется, модалка закрывается
    И SCR-006 показывает «Спасибо за оценку» и поставленную оценку в режиме чтения

  Сценарий: AC-002 — оценка без комментария
    Дано клиент выбрал звёзды, не заполнив комментарий
    Тогда отправка проходит успешно

  Сценарий: AC-003 — отказ от оценки сейчас
    Когда клиент нажимает «Не сейчас»
    Тогда модалка закрывается без отправки
    И кнопка «Оценить инструктора» остаётся доступной позже
```

## 8. Трассировка
| Артефакт | Связь |
|---|---|
| FR-40 | Разовая оценка 1–5 звёзд + комментарий, без редактирования. |
| FR-41 | Влияет на средний рейтинг, отображаемый в SCR-002/SCR-003. |
| UC-4 | Полный сценарий оценки инструктора. |
