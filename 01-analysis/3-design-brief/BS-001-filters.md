# Требования на дизайн · BS-001 · Фильтры

**ID:** BS-001 · **Тип:** Модалка (bottom sheet / диалог) · **Зона:** АЗ · **Приоритет:** High · **Статус:** Черновик · **Версия:** 0.1 · **Дата:** 2026-07-04

**Источники:** [Foundations](00-foundations.md) · [ФТ](../functional-requirements.md)

---

## 1. Назначение и контекст
Позволяет клиенту сузить список тренировок на [SCR-002](SCR-002-slot-list.md) по дате, зоне/формату и инструктору (FR-11).

## 2. Навигация
- Входящая: кнопка «Фильтры» на SCR-002.
- Исходящая: применение/сброс → закрытие → SCR-002 с обновлённым списком.

## 3. Контент-инвентарь
| # | Элемент | Тип | Примечание |
|---|---|---|---|
| 1 | Период дат (от/до) | date range | Дефолт — 7 дней, включительные границы, локальная таймзона |
| 2 | Зона/формат | множественный выбор (чекбоксы) | «Для новичков» / «Опытные» |
| 3 | Инструктор | множественный выбор | Список из справочника; если пуст — блок скрыт |
| 4 | Кнопка «Сбросить» | текстовая кнопка | Очищает все фильтры |
| 5 | Кнопка «Показать N тренировок» | основной CTA | N — live-счётчик по текущим условиям (если считается на клиенте/легко получить с бэкенда) |

## 4. Структура
```
┌─────────────────────────────┐
│  ▭▭▭        Фильтры      ✕  │
│  Период                      │
│  [ от: 07.07 ] [ до: 14.07 ] │
│  Формат                      │
│  ☐ Для новичков  ☐ Опытные   │
│  Инструктор                  │
│  ☐ Анна   ☐ Игорь  ☐ ...     │
│  ┌───────────┐ ┌───────────┐ │
│  │ Сбросить  │ │ Показать  │ │
│  └───────────┘ └───────────┘ │
└─────────────────────────────┘
```

## 5. Поведение
- Внутри группы (формат, инструктор) — комбинирование по «ИЛИ»; между группами — по «И».
- Изменения применяются по кнопке «Показать», не мгновенно (чтобы не дёргать список при каждом клике).
- «Сбросить» возвращает дефолт (без фильтров, 7 дней) и сразу закрывает модалку с обновлённым списком.

## 6. Состояния
Модалка не делает отдельных сетевых запросов на загрузку себя — справочник инструкторов может подгружаться (Loading → Content/Error по общему паттерну, foundations §5).

## 7. Критерии приёмки
```gherkin
Функция: Фильтрация тренировок (BS-001)

  Сценарий: AC-001 — применение фильтров
    Дано клиент выбрал формат «Для новичков» и инструктора «Анна»
    Когда нажимает «Показать»
    Тогда SCR-002 показывает только тренировки формата «Для новичков» ИЛИ с инструктором «Анна»... 
    (уточнение: внутри группы ИЛИ, между группами И — см. §5)

  Сценарий: AC-002 — сброс фильтров
    Дано применены какие-либо фильтры
    Когда клиент нажимает «Сбросить»
    Тогда список возвращается к дефолтному состоянию (7 дней, без фильтров)

  Сценарий: AC-003 — пустой справочник инструкторов
    Дано список инструкторов пуст
    Тогда блок «Инструктор» не отображается
```
